# AMPLE.py

This repository contains code and other files relating to the python version of AMPLE.

The code is based on the original AMPLE which was written in MATLAB by Will Coombs and is described in this paper 

[*Coombs, W.M. and Augarde, C.E. (2020) 'AMPLE : A Material Point Learning Environment.', Advances in engineering software., 139. p.102748*](https://www.sciencedirect.com/science/article/pii/S0965997819304648)

Further information can be found at  the [AMPLE](https://wmcoombs.github.io/) webpage.

The code and [associated Jupyter Notebook](../AMPLE-Py.ipynb) which you will find here are the results of final year project work undertaken at the Department of Engineering, Durham University by [Ellie Bacon](https://www.linkedin.com/in/eleanor-bacon/) in 2021.
